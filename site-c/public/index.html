なぜビジターIDが引き継がれないのか？

  1. Site C への遷移リンクに adobe_mc パラメータが付与されていない
  2. Site C の設定で crossDomainEnabled: false になっている
  3. そのため、Site C では新しいビジターIDが生成される
  4. 結果として、Site A/B での行動と Site C での行動は別々の訪問者として記録される
